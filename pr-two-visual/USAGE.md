# Interactive Set Operations Analyzer

This interactive script analyzes the ProjectTwo C# program and creates visual representations.

## Features

- **Interactive Venn Diagram**: Dynamically visualize set operations
- **Project Analysis**: Parse and display project metadata
- **Multiple Visualizations**: Generate both diagrams and statistical views

## Usage

```bash
python interactive_analyzer.py
```

## Generated Output

Creates PNG images showing:
1. Venn diagrams for all three sets
2. Step-by-step operation visualization
3. Final result highlighting
4. Project structure information
